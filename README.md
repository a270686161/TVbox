# 📺 TVBox 接口说明

欢迎使用 TVBox 自定义接口！

TVBox 是一款开源的 IPTV 播放器，支持通过接口加载各种源，包括直播、点播、聚合平台等内容。通过本项目，你可以获取专为 TVBox 定制的稳定接口。

---

## 🚀 功能特色

- 聚合多种资源：直播、影视、动漫、综艺等
- 高速稳定：每日自动维护更新
- 持续优化：接口内容不断完善，持续更新

---

## 🧩 使用方法

1. 打开 TVBox 应用
2. 进入设置界面
3. 将接口地址粘贴至“自定义接口”位置
4. 点击“确定”并返回，即可加载内容
